### Setup

#### `.env` requirements

- `PORT` - Port Number
- `MONGODB_URI` - URL to the running mongo instance/db
- `SECRET` - to generate accses token
- `CLIENT_ID`
- `CLIENT_SECRET`
- `API_SERVER`


#### Running the app

- `npm start`
- Endpoint: `/` 
- Endpoint: `/signup` >> POST
- Endpoint: `/signin` >> POST
- Endpoint: `/users` >> GET
- Endpoint: `/oauth` >> GET
- Endpoint: `/secret` >> GET
- Endpoint: `/add` >> POST
- Endpoint: `/change` >> PUT
- Endpoint: `/read` >> GET
- Endpoint: `/remove` >> DELETE
